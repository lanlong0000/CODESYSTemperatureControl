### PAC控制器实现温度控制
运用日本康泰克CONTEC公司的PAC控制器Conprosys CPS-PC341MB，远程I/O（CPS-Nano）等，能熟练运用CODESYS，NI LabVIEW，NI OPC Server等工具，能完成从设计，开发，仿真，实现等一系列，完成PAC开发，实现温度自动过程比例积分控制及手动调控。
